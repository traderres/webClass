Problem:  In the IntelliJ Project View, it no longer shows the parent of the project
------------------------------------------------------------------------------------


Solution #1:  Re-import the parent
----------------------------------
 1. Pull File -> Project Structure

 2. Click on Modules

 3. Click on +

 3. Select Import Module

 4. Browse to the root of the project folder

 5. Select it
    -- Then, Intellij shows the parent project



Solution #2:  Erase the .idea/ directory and restart
