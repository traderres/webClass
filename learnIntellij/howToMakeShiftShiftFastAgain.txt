Problem:  Shift-shift is really slow in Intellij 2023.2.5
Solution: Fix it


Procedure
---------
 1. Startup IntelliJ
 2. Pull File -> Settings
 3. Search for "Wait for all"
 4. Click on "Advanced Settings"
 5. Uncheck the "Wait for all contributors to finish before showing results"
 6. Press OK

Now, shift-shift is fast again!
