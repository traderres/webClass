How to Tell Java Program to Trust a Cert
----------------------------------------
Problem:  My java program does not trust this url:  https://icamdev.dcsa.mil/
Solution: Get the cert and add it to the trust

Assumptions
 A) You have a trust file here:
    /tmp/cacerts.all


Procedure
---------
 1. Use openssl to get the cert info
    unix> openssl s_client -showcerts -connect icamdev.dcsa.mil:443

 2. Verify that the cert is valid
    unix>  openssl x509 -in   /tmp/icamdev1.crt -text

 3. Add this cert to your existing trust
    unix> keytool -import -alias "my-alias-or-something" -keystore /tmp/cacerts.all  -file /tmp/icamdev1.crt

 4. Restart your java process
    NOTE:  It should have this
              -Djavax.net.ssl.trustStore/tmp/cacerts.all -Djavax.net.ssl.trustStorePassword=changeit
