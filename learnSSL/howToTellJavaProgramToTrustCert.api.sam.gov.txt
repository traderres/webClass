How to Tell Java to trust api.sam.gov using TLS
-----------------------------------------------
 1. We use openssl s_client to get the external site's certificate
    Use openssl to connect to https://api.sam.gov
    TERMINAL> openssl s_client -connect api.sam.gov:443
