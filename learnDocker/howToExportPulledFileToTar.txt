How to Export a Pulled File to a TAR file
-----------------------------------------
Problem:  I need to export a version of keycloak into a TAR
Solution: Use docker save


Procedure
---------
 1. Pull the image
    unix> docker pull registry1.dso.mil/ironbank/opensource/keycloak/keycloak:26.2.4

 2. Export the image to a file
    unix> cd /tmp
    unix> docker save -o keycloak.tar  registry1.dso.mil/ironbank/opensource/keycloak/keycloak:26.2.4
