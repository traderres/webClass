How to Stop a Long-Running Materialized View Refresh
----------------------------------------------------

Procedure
---------
 1. Login as the same user that is refreshing the materialized view

 2. Find the refresh pid
	SELECT pid, datname, usename, client_addr, state, query
	FROM pg_stat_activity
	WHERE query ILIKE '%REFRESH MATERIALIZED VIEW%';

 3. Terminate the process:
    SELECT pg_terminate_backend(your_pid_here);
