How to Install Maven 3.9.5 on Windows
--------------------------------------

Assumptions
 A) You have a Java JDK installed


Procedure
---------
 1. Download Maven apache-maven-3.9.5-bin.zip
    a) Go to https://maven.apache.org/download.cgi
    b) Look for apache-maven-3.9.5-bin.tar.zip
       *OR*
       Go to https://dlcdn.apache.org/maven/maven-3/3.9.5/binaries/apache-maven-3.9.5-bin.zip

    c) Save apache-maven-3.9.5-bin.tar.gz to your Downloads/ directory


 2. Install Maven to c:\tools\apache-maven-3.9.5\
    a. Right-click on c:\vault\apache-maven-3.9.5-bin.zip -> 7-Zip -> Extract files....
    b. In the "Extract" popup window
          Under Extract To:  c:\tools
          Press OK

          Now, you should have c:\tools\apache-maven-3.9.5\


 3. Create an environment variable called M2_HOME=c:\tools\apache-maven-3.9.5
    a. Open the Environment Variables in Windows by pressing <Start><Run>environment

    b. Under "User variables for..." [on the top of this popup],
       Click "New..."
                Variable name:  M2_HOME
                Variable value: c:\tools\apache-maven-3.9.5


  4. Add the Maven\bin directory to your PATH
     a. Open the Environment Variables in Windows by pressing <Start><Run>environment

     b. Under "User variables for..." [on the top of this popup],
        Click "New..."  [if PATH already exists, then double-click on PATH and add a semicolon and append this to the end of the PATH]
                Variable name:  PATH
                Variable value: c:\tools\apache-maven-3.9.5\bin

  5. Verify that Maven works
     a. Press <Start><Run>CMD
        DOS> mvn -version

        You should see the following:
              Apache Maven 3.9.5 (57804ffe001d7215b5e7bcb531cf83df38f93546)
              Maven home: c:\tools\apache-maven-3.9.5
