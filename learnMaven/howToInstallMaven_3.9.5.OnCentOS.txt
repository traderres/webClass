How to Install Maven 3.9.5 on CentOS
------------------------------------

Assumptions
 A) You have a Java JDK installed


Procedure
---------
 1. Download Maven apache-maven-3.9.5-bin.zip
    a) Go to https://maven.apache.org/download.cgi
    b) Look for apache-maven-3.9.5-bin.tar.gz
       *OR*
       Go to https://archive.apache.org/dist/maven/maven-3/3.9.5/binaries/apache-maven-3.9.5-bin.tar.gz

    c) Save apache-maven-3.9.5-bin.tar.gz to your /tmp


 2. Install maven on your CentOS box
    unix> sudo -s
    unix> cd /opt
    unix> cp /tmp/apache-maven-3.9.5-bin.tar.gz  .
    unix> tar zxvf apache-maven-3.9.5-bin.tar.gz
    unix> rm apache-maven-3.9.5-bin.tar.gz
    unix> exit


 3. Add the mvn command to your PATH by adjusting your .bashrc
    unix> vi ~/.bashrc

       # Add these to the end of your .bashrc
       export M2_HOME=/opt/apache-maven-3.9.5
       export PATH=${M2_HOME}/bin:${PATH}


 4. Verify that maven works
    a. Open a new terminal
    b. In the terminal, run this command:
       unix> mvn -version
