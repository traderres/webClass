How to Install Clementime Music Player on Rocky 9
------------------------------------------


Procedure
---------
 1. Follow the procedures to install snap on Rocky 9
    [See howToInstallSnapOnRocky9.txt]

 2. Use snap to install clementine
    unix> sudo snap install clementine

 3. Run clementine
    unix> clementine

    NOTE:  The first time you start, it can take 60 seconds to setup the initial database
