How to Make Firefox the Default Browser
---------------------------------------
Problem:  Google is my default browser and I want firefox to be it


Procedure to set Firefox as Default Browser
-------------------------------------------
 1. Create a firefox.desktop entry
    unix> vi /tmp/firefox.desktop

	[Desktop Entry]
	Type=Application
	Name=firefox2
	Comment=firefox web browser
	Terminal=false
	Exec=firefox
	Categories=Network;WebBrowser


 2. Install the firefox.desktop file
    unix> cd /usr/share/applications
    unix> sudo cp /tmp/firefox.desktop .

 3. Tell rocky linux to use firefox as the default browser
    unix> xdg-settings set default-web-browser firefox.desktop

 4. Verify it works
    xdg-open https://www.google.com/
    -- It should open a firefox browser



Procedure to set Google Chrome as Default Browser
-------------------------------------------------
 1. Find the google chrome entry in /usr/share/applications
    unix> cd /usr/share/applications
    unix> ls *goo*
    google-chrome-beta.desktop

 2. Tell rocky linux to use Google Chrome as the default browser
    unix> xdg-settings set default-web-browser google-chrome-beta.desktop

 4. Verify it works
    xdg-open https://www.google.com/
    -- It should open a google chrome browser
