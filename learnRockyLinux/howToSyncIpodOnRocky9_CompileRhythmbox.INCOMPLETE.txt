How to Compile Rhythm Box from Source Code for Rocky 9
-------------------------------------------------------
Problem:  I want an iTunes replacement but none exist in Linux
Solution: Compile rhythmbox from scratch



Assumptions
 A) You have a classic IPOD that you want to sync up


Procedure
---------
 1. Install dependencies
    b. Verify the gcc compiler
       unix> gcc --version
       gcc (GCC) 11.4.1 20230605 (Red Hat 11.4.1-2)

    b. Install Meson and Ninja
       unix> sudo dnf install meson ninja-build gdk-pixbuf2-devel gobject-introspection-devel gstreamer1-devel gstreamer1-plugins-base-devel \
                              cmake gtk3-devel json-glib-devel libpeas-devel libsoup-devel clang glib2-devel libnghttp2-devel sqlite-devel libpsl-devel


 2. Compile lib-soup-3
    unix> cd ~/intellijProjects
	unix> git clone https://gitlab.gnome.org/GNOME/libsoup.git
	unix> cd libsoup/

	unix> vi meson.build

	       # Change this line to this:     (so it it works with my version of glib)
	       glib_required_version = '>= 2.68.4'

    unix> meson _build -Dprefix=$PWD/_install          # Prepare the build
    unix> ninja -C _build                              # build lib-soup-3
    unix> ninja -C _build install                      # install lib-soup-3


 2. Download rhythmbox 3.4.7
    unix> cd ~/Downloads
    unix> wget https://github.com/GNOME/rhythmbox/archive/refs/tags/v3.4.7.tar.gz
    unix> tar zxvf v3.4.7.tar.gz
    unix> rm v3.4.7.tar.gz

 3. Compile from source code
    unix> cd rhythmbox-3.4.7/
    unix> meson _build -Dprefix=$PWD/_install          # Prepare the build
    unix> ninja -C _build                              # build Rhythmbox
    unix> ninja -C _build install                      # install Rhythmbox
