How to Install Postman on Rocky 9
---------------------------------



Procedure
---------
 1. Follow the procedures to install snap on Rocky 9
    [See howToInstallSnapOnRocky9.txt]

 2. Use snap to install postman
    unix> sudo snap install postman

 3. Run postman
    a. Open a new terminal
    b. In the terminal
       unix> postman

 4. In the welcome screen, press "Continue without an account"

 5. In the "Unlock Postman with an account", press "Open Lightweight API Client"


 6. Create a shortcut in the MATE Window Manager
    a. Download the icon
       1) Go to https://uxwing.com/postman-icon/
       2) Press "Download PNG"
       3) Save it to your Downloads

    b. Right-click on the panel -> Add to Panel
    c. Click on "Custom Application Launcher" -> Add
    d. In the "Create Launcher"
       Name:  	 Postman
       Command:  /var/lib/snapd/snap/bin/postman
       Icon:     Browse to Downloads/postman-icon.png



Troubleshoot / How to fix strange characters in Postman dialog
---------------------------------------------------------------
have the same issue.. had fixed it successfully yet. Here is step:
See https://github.com/postmanlabs/postman-app-support/issues/10449
    https://github.com/postmanlabs/postman-app-support/issues/10688

 1. update font
    unix> sudo fc-cache -r -v

 2. find font config of snap application (postman):
    unix> find ~/snap/postman/ -name 'fontconfig'
    /home/adam/snap/postman/common/.cache/fontconfig
    /home/adam/snap/postman/350/.config/fontconfig
    /home/adam/snap/postman/360/.config/fontconfig

 3. Remove the above directories
    unix> rm -rf /home/adam/snap/postman/common/.cache/fontconfig
    unix> rm -rf /home/adam/snap/postman/350/.config/fontconfig
    unix> rm -rf /home/adam/snap/postman/360/.config/fontconfig

 4. remove all cache font config in the result from above:
    rm -rf ~/snap/postman/common/.cache/fontconfig/
    rm -rf ~/snap/postman/common/.cache

 4. Reboot
    unix> reboot

 5. Start up Postman
    Pull File -> Settings -> Certificates -> "Add Certificate"
    Press "Select File"
    --> There may be a 2 minute DELAY while the cache is reloaded
        Then, the dialog should show up correctly
