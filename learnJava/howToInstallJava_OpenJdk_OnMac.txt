How to Install OpenJDK on a Mac
-------------------------------
Red Hat builds of OpenJDK is *NOT* available for Mac.
However, there’s Temurin, a version of OpenJDK that Red Hat actively contributes to that’s available for Mac.


Red Hat has provided notes here:
   https://developers.redhat.com/products/openjdk/downloads/macos


How to install OpenJDK 17 for Mac using Temurin
------------------------------------------------
Install OpenJDK 17 using Homebrew and Temurin
unix> brew tap homebrew/cask-version
unix> brew install --cask temurin17


How to install OpenJDK 11 for Mac using Temurin
unix> brew tap homebrew/cask-version
unix> brew install --cask temurin11
