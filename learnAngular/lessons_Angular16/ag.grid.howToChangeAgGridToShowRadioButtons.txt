How to Get ag-grid to use radio buttons instead of check boxes  (handy for single-select rows)
----------------------------------------------------------------------------------------------
Problem:  The user can only select one row but ag-grid shows a checkbox
Solution: Apply the single-select class to change the grid's checkboxes to radio buttons



Procedure
---------
 1. Add this to your project's styles.scss

    // Create the radio button object
    .single-select .ag-body .ag-checkbox-input-wrapper {
      width: 16px;
      height: 16px;
      border: 1.5px solid #666;
      border-radius: 50%;
      background-color: white;
      position: relative;
    }

    // Hide the native checkbox object by wiping the content field it's placed inside of
    .single-select .ag-body .ag-checkbox-input-wrapper::after {
      content: "";
    }

    // Position the 'dot' (fill of the radio button) inside the button (circle)
    .single-select .ag-body .ag-row.ag-row-selected .ag-checkbox-input-wrapper::after {
      content: "";
      position: absolute;
      top: 50%;
      left: 50%;
      width: 8px;
      height: 8px;
      background-color: #666;
      border-radius: 50%;
      transform: translate(-50%, -50%);
    }


 2. Apply the single-select class


     <ag-grid-angular
          style="width: 100%; height: 100%;"
          class="ag-theme-balham"
          [gridOptions]="this.gridOptions"
          [defaultColDef]="this.defaultColDefs"
          [class.single-select]="gridOptions.rowSelection === 'single'"       <-- Apply the "single-select" class if the grid uses single-select rowSelection
          (selectionChanged)="onSelectionChanged($event)"
          (firstDataRendered)="this.firstDataRendered()"
          [columnDefs]="this.columnDefs"
          (gridReady)="this.onGridReady($event)">
        </ag-grid-angular>
