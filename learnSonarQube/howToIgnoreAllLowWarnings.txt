How to Ignore All Low Warnings in IntelliJ
------------------------------------------
Problem:  I want the "SonarQube for IDE" plugin to ignore the "Low" warnings
Solution: Create a solarlint.xml file and install it


Procedure
---------
 1. Download the solarlint.xml file


 2. Close IntelliJ

 3. Install it to this location:
    Linux:
    	a. Search where your solarlint.xml file is located
    	   unix> cd ~
    	   unix> find . -type f -iname 'sonarlint.xml' -print
    	   ./.config/JetBrains/IntelliJIdea2023.2/options/sonarlint.xml


		b. Overwrite this file with the the downloaded solarlint.xml
		   unix> cp ~/Downloads/solarlint.xml   ./.config/JetBrains/IntelliJIdea2023.2/options/sonarlint.xml



	Windows:
		a. Search where your solarlint.xml file is located
		   terminal> dir /s solarlint.xml
		   c:\users\MY_USERNAME\AppData\Roaming\JetBrains\IntelliJIdea2024.3\option\solarlint.xml

		b. Overwrite this file with the the downloaded solarlint.xml
		   terminal> copy PATH\To\Downloads\solarlint.xml  c:\users\MY_USERNAME\AppData\Roaming\JetBrains\IntelliJIdea2024.3\option\solarlint.xml


 4. Startup IntelliJ
    -- Now, when you run a SonarQube scan it will ignore all of "Low Priority" warnings
