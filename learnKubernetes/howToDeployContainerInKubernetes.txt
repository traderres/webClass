How to Deploy Docker Container in Kubernetes
--------------------------------------------

Assumptions:
 A) You have MiniKube installed
 B) You followed the steps in howToCreateContainerAndPushItToRepo.txt

Procedure
---------
 1. Verify that minikube is working
	unix> minikube start          # do not run sa root
	unix> kubectl cluster-info

	Kubernetes control plane is running at https://192.168.49.2:8443
	CoreDNS is running at https://192.168.49.2:8443/api/v1/namespaces/kube-system/services/kube-dns:dns/proxy

 2. List add ons
    unix> minikube addons list

 3. Connect to the minikube dashboard
    unix> minikube dashboard &      # This will open a browser to the dashboard


 4. Deploy the kubia
    a. List images
       unix> docker images

    b. Use kubernetes to run one of them
       unix> kubectl run kubia --image=traderres/my-public-repo:kubia-1.0  --port=8080 --image-pull-policy=IfNotPresent

	   You should see the following:
    	   pod/kubia created


  5. Get the pod status
     unix> kubectl get pods

     You should see the following:
		NAME    READY   STATUS              RESTARTS   AGE
		kubia   0/1     ContainerCreating   0          9s


  6. Get the pod status   (30 seconds later -- it should show "Running")
     unix> kubectl get pods

     You should see the following:
		NAME    READY   STATUS    RESTARTS   AGE
		kubia   1/1     Running   0          32s
