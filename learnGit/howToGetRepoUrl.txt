How to get the Git Repo URL from Command-line
---------------------------------------------

Procedure
---------
unix> git remote get-url origin
-- WIll show you the url



Example
-------
unix> git clone https://github.com/traderres/webClass.git
unix> cd webClass
unix> git remote get-url origin
git@github.com:traderres/webClass.git

If you connect to github.com/traderres and the repo is public, you will see it
