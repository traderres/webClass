How to Cut a Release
--------------------

ASSUMPTIONS:
 A) You want to create a release of version 2.0.0
 B) You want the next version to be         2.0.1-SNAPSHOT


Procedure
---------
 1. Unprotect the development branch in Gitlab  (so we can make changes directly to development)
    a. Connet to your gitlab repository
    b. On your Project -> Settings Repository
    c. Under "Protected Branches" press "Expand"
    d. Press the "Unprotect" next to the development branch

 2. Get the latest development branch
    unix> cd ~/intellijProjects/MY_PROJECT_NAME
    unix> git branch -D development
    unix> git fetch -a
    unix> git checkout -f development

 3. Set the new version to be 2.0.0
    unix> mvn versions:set -DnewVersion=2.0.0  -DgenerateBackupPoms=false
    unix> git commit -a -m "Set version to 2.0.0"

 4. Make the build
    unix> mvn clean package -Pprod

 5. Deliver the build to the obss

 6. Create a tag called "2.0.0"
    unix> git tag -a 2.0.0   -m "Created a tag for build 2.0.0"
    unix> git push --tags

 7. Set the new version to be 2.0.1-SNAPSHOT
    unix> mvn versions:set -DnewVersion=2.0.1-SNAPSHOT  -DgenerateBackupPoms=false
    unix> git commit -a -m "Set version to 2.0.1-SNAPSHOT"

 8. Push the changes up
    unix> git push -f

 9. Protect the development branch in Gitlab
      a. Connet to your gitlab repository
      b. On your Project -> Settings Repository
      c. Under "Protected Branches" press "Expand"
      d. In the "Protet a Branch" section
         Branch Name:  development
         Allowed to merge:       Developers + Maintainers
         Allowed to push:        No One
         Allowed to force push:  <not selected>
      e. Press "Protect"
