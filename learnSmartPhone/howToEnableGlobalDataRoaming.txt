How to Enable Global Data Roaming (Android Phone)
-------------------------------------------------
You will want to make sure your Androidâ€™s roaming settings are on, so you can connect with the local cell towers.

NOTE:  With Verizon you have to sign-up for TravelPass and then you can use global roaming for $10/day.

Procedure
---------
 1. In your phone, select Settings
 2. Select More > Mobile Networks
 3. Select Global Data Roaming Access or Data Roaming Access 
 4. Select Allow access for all trips.
