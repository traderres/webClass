How to Stop Firefox from Asking to be the Default Browser on Startup
--------------------------------------------------------------------


Procedure
---------
 1. Startup firefox
 2. In the URL, type-in:  about:config
 3. Press the blue button to proceed
 4. Search for checkDefaultBrowser
     -- Double-click on browser.shell.checkDefaultBrowser so it's FALSE

If set to FALSE, then firefox will not ask to be the default browser on startup
